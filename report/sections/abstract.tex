\begin{abstract}
Neste documento é apresentada uma análise de códigos para codificadores de canais baseados no código de Hamming. Para isso um sistema de comunicações foi simulado utilizando um canal BSC com parâmetro $p$ e um método de geração de códigos foi proposto. Por fim foi realizada a comparação dos códigos gerados com o de Hamming.
\end{abstract}
